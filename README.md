# Restaurant SQL Analysis

This project contains SQL queries to analyze restaurant data.

## Analysis Covered
1. Total countries with restaurants
2. Total cities with restaurants
3. Total restaurants
4. Total votes
5. Average restaurant rating
6. Restaurants opened by:
   - Year
   - Month
   - Quarter
7. Restaurants by city
8. Restaurants by country
9. Online delivery percentage
10. Table booking percentage
11. Restaurants by price bucket
